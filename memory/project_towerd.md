A browser-based medieval tower defense game where a king defends his castle against enemy waves.

**Core mechanics:**
- Place and upgrade soldiers (towers) along enemy path
- Kill enemies to collect coins
- Coins used to place new soldiers or upgrade existing ones (3 tiers)
- Survive increasingly difficult waves; boss every 5th wave

**Soldier types planned:** Archer, Crossbowman, Mage, Knight, Catapult
**Enemy types planned:** Goblin, Orc, Troll, Dark Knight, Dragon (boss)

**Status:** PRD created. Tech stack and map/placement system TBD (open questions in PRD.md).

**Why:** User's personal game project, starting from scratch.
**How to apply:** Reference PRD.md for decisions. When building, ask about open questions (placement system, path style, art style, tech stack) before coding.
